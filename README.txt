----------------
Robot Framework
----------------

INSTALL
Include RobotFramework.jar in your program
Additional Libraries:
If you plan on using components from ComputerDevices you must include lib/jna.jar and lib/platform.jar
If you plan on using components from RobotDashboard you must include lib/guava-12.0.jar
To use the RXTX serial library you must include lib/RXTXcomm.jar and specify its Native Library location to be lib/rxtx-2.1-7-bins-r2/YOUR_MACHINE_TYPE
